#### F201: __tests__/ 注記セクションの記述補強

**カテゴリ**: 明確性

**問題**:
影響規模テーブルでは「src/lib/__tests__/ 9ファイル中6ファイル」と記載されているが、__tests__/ 注記セクションの方では影響6ファイルの具体名のみで「9ファイル中」の情報がない。

**推奨対応**:
注記セクション冒頭に「9ファイル中6ファイルが移動対象ファイルへの相対importを持つため更新が必要:」のように追記する。

#### F202: db/ グループ内部依存の補足

**カテゴリ**: 完全性

**問題**:
推奨移行順序で db/ が「他グループへの依存がなく独立している」と記載されているが、グループ内部の依存構造（db-repository -> db-instance -> db 等）の簡単な説明があるとより明確。

**推奨対応**:
必須ではないが、db/ 内部の依存方向を簡潔に補足すると独立性の根拠がより明確になる。
