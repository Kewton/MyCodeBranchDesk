# Security Policy

## 報告方法

セキュリティに関する問題を発見した場合は、**公開 Issue を作成しないでください**。

以下の方法で報告をお願いします:

- GitHub の [Security Advisories](https://github.com/Kewton/MyCodeBranchDesk/security/advisories) から非公開で報告
- または、リポジトリオーナーに直接連絡

## 報告に含めてほしい情報

- 問題の概要
- 再現手順
- 影響範囲の見積もり
- 可能であれば修正案

## 対応方針

- 報告を受領後、可能な限り速やかに確認します
- 修正が必要な場合、修正版をリリースした後に詳細を公開します
- 報告者のクレジットは、ご希望に応じて記載します

## 対象バージョン

最新のリリースバージョンのみをサポート対象とします。

## セキュリティに関する設計方針

本ツールのセキュリティモデルについては [Trust & Safety](./docs/TRUST_AND_SAFETY.md) を参照してください。
