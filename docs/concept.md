# CommandMate - Concept

## Vibe Coder のための並行開発プラットフォーム

> **「仕事も、育児も、家事も。その隙間で、コードを紡ぐ。」**

---

## このツールが解決する課題

### Vibe Coding の現実

AI コーディングアシスタント（Claude Code など）の登場により、開発のあり方は大きく変わりました。いわゆる「Vibe Coding」— AI との対話を通じて直感的にコードを書くスタイル — は、多くの開発者に新しい可能性をもたらしています。

しかし、ターミナルだけの開発には限界があります：

- **場所に縛られる**: PC の前に座らないと開発できない
- **時間が必要**: まとまった時間がないと作業を始められない
- **コンテキストスイッチのコスト**: 1つのタスクに集中すると、他のタスクが止まる
- **生活との両立が難しい**: 仕事、育児、家事の合間に開発する余裕がない

### 私たちの答え

CommandMate は、これらの課題に対する私たちの答えです：

```
🚀 並行開発 × 📱 モバイルアクセス × 📦 1つのアプリで完結 = ∞ 生産性
```

複数のツールを行き来する必要はありません。**ブラウザ1つで、指示出し・実行確認・履歴管理がすべて完結**します。

---

## コンセプト

### 1. ブランチごとの独立した開発環境

Git worktree の概念を活用し、**ブランチごとに独立した AI セッション**を管理します。

```
feature/authentication  →  Claude Code セッション A
feature/dashboard       →  Claude Code セッション B
bugfix/login-error      →  Claude Code セッション C
```

これにより：
- **複数タスクの並行処理**: AI が作業中でも、別ブランチの作業を開始できる
- **コンテキストの保持**: 各ブランチの会話履歴が独立して保存される
- **柔軟な作業スタイル**: 気分や状況に応じてタスクを切り替えられる

### 2. スマートフォンからの開発

**PC がなくても開発を進められる**。これが CommandMate の核心です。

- 通勤電車の中で PR レビューの指示を出す
- 子どもが寝ている間にバグ修正を依頼する
- 家事の合間にテストの実行結果を確認する

スマホ最適化された UI により、**隙間時間を開発時間に変換**できます。

### 3. 非同期・イベント駆動

AI の応答を待つ必要はありません。

1. スマホからタスクを指示
2. ポケットにしまって日常に戻る
3. AI が完了したら通知を受け取る
4. 結果を確認して次の指示を出す

このサイクルにより、**AI の処理時間を生活時間として活用**できます。

## 対象ユーザー

CommandMate は、以下のような方に最適です：

### 👨‍👩‍👧‍👦 子育て中のエンジニア

> 「まとまった時間は取れないけど、5分10分の隙間時間ならある」

子どもが遊んでいる間、昼寝中、就寝後の短い時間を積み重ねて開発を進められます。

### 👔 本業を持つ副業開発者

> 「平日は仕事、週末も家族との時間。でも自分のプロジェクトも進めたい」

通勤時間、昼休み、待ち時間を活用して、本業に影響を与えずに開発できます。

### 🏠 リモートワーカー

> 「自宅での作業は集中が難しい。家事との切り替えもストレス」

家事の合間にタスクを投げ、終わったら結果を確認。生活のリズムを崩さずに開発できます。

### 🌟 Vibe Coder

> 「AI との対話で開発するのは楽しい。でもターミナルに張り付くのは疲れる」

チャット形式の UI で、より自然に AI と協働できます。

## 実現する生産性向上

### Before: 従来の開発スタイル

```
[PC作業] ━━━━━━━━━━━━━━━━━━━━━━━━━━━
          ↑                            ↑
       開発開始                     開発終了
       (2時間の集中時間が必要)
```

### After: CommandMate スタイル

```
[通勤] [仕事] [昼休] [仕事] [帰宅] [家事] [育児] [就寝前]
  📱     💼    📱     💼     🚃     🏠     👶      📱
  ↑            ↑                          ↑        ↑
タスク指示  結果確認              子守りしながら  レビュー
(3分)      (5分)                 結果チェック   (10分)
                                  (2分)
```

**分散した隙間時間の合計が、まとまった開発時間に匹敵する成果を生む。**

## 主要機能

| 機能 | 説明 |
|------|------|
| **マルチワークツリー管理** | ブランチごとに独立した AI セッション |
| **スマホ最適 UI** | タッチ操作に最適化されたチャットインターフェース |
| **リアルタイムステータス** | AI の状態（処理中/待機中/完了）を即座に把握 |
| **CLI サポート** | Claude Code |
| **ログ管理** | 詳細な会話履歴を Markdown で保存 |
| **LAN アクセス** | 同一ネットワーク内のスマホからアクセス可能 |

## 技術スタック

- **フロントエンド**: Next.js 14 (App Router) / TypeScript / Tailwind CSS
- **バックエンド**: Next.js API Routes / Node.js
- **データベース**: SQLite
- **セッション管理**: tmux
- **リアルタイム通信**: WebSocket
- **対応 CLI**: Claude Code

## 始め方

```bash
# インストール
git clone https://github.com/Kewton/CommandMate.git
cd CommandMate
npm install

# 設定（.env）
CM_ROOT_DIR=/path/to/your/repos
CM_BIND=0.0.0.0
CM_AUTH_TOKEN=your-secret-token

# 起動
npm run build && npm start
```

スマホのブラウザから `http://<your-pc-ip>:3000` にアクセスして開始。

詳細は [README.md](../README.md) を参照してください。
