## 問題の詳細

### 症状
- マークダウンプレビューでコードブロックがスタイリングされない
- 番号付きリスト（`1.`）が正しく解釈されない
- その他マークダウン要素（見出し、引用、テーブル等）のスタイルが欠落

### 根本原因
`MarkdownEditor.tsx` で使用している `prose` クラスは `@tailwindcss/typography` プラグインが提供するものだが、このプラグインがプロジェクトにインストールされていなかった。

**影響箇所:**
- `src/components/worktree/MarkdownEditor.tsx:291`
  ```tsx
  className="flex-1 p-4 overflow-y-auto prose prose-sm max-w-none"
  ```

## 修正内容

### 変更ファイル

| ファイル | 変更内容 |
|----------|----------|
| `package.json` | `@tailwindcss/typography` を devDependencies に追加 |
| `tailwind.config.js` | plugins 配列に `require('@tailwindcss/typography')` を追加 |

### 具体的な変更

**tailwind.config.js:**
```javascript
// Before
plugins: [],

// After
plugins: [
  require('@tailwindcss/typography'),
],
```
