#### D2-001: updateWorktreeLinkがsession-db.tsに誤配置

| 項目 | 内容 |
|------|------|
| **カテゴリ** | 設計書-コード整合 |
| **場所** | セクション4-1 db.ts分割設計 |

セクション4-1のsession-db.tsの主要関数リストに`updateWorktreeLink`が含まれている。しかし実際のコードでは、この関数はworktreesテーブルのlink列を更新する処理であり、セッション状態管理とは無関係である。

**実コード** (db.ts 435-447行):
```typescript
export function updateWorktreeLink(
  db: Database.Database,
  worktreeId: string,
  link: string
): void {
  const stmt = db.prepare(`
    UPDATE worktrees
    SET link = ?
    WHERE id = ?
  `);
  stmt.run(link || null, worktreeId);
}
```

**改善提案**: `updateWorktreeLink`をworktree-db.tsの主要関数リストに移動し、行数目標を調整する。
